# Internship Logos

Place your internship company logo files in this directory. 
Supported formats: `.png`, `.jpg`, `.jpeg`, `.svg`

The files should be named to match the `logoKey` defined in `src/routes/index.tsx`:
* Plasticure Private Limited -> `plasticure.png` (or `.jpg` etc.)
* Central Tool Room & Training Centre -> `cttc.png`
* Tata Group (Forage) -> `tata.png`
* Google Cloud Skills Boost -> `gcp.png`
* Parala Maharaja Engineering College -> `pmec.png`
* ROBOSTREAKS Robotics Club -> `robostreaks.png`
